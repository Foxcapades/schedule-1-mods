= Schedule I Mods

A collection of tiny mods to tweak to the gameplay experience.

== Mods

=== link:mods/AdjustableDayLength/[Adjustable Day Length]

Adds a configurable multiplier value to adjust game day length.

[none]
* On Nexus Mods: https://www.nexusmods.com/schedule1/mods/1799
* Thunderstore (Il2Cpp): https://new.thunderstore.io/c/schedule-i/p/Foxcapades/AdjustableDayLengthIl2Cpp
* Thunderstore (Mono): https://new.thunderstore.io/c/schedule-i/p/Foxcapades/AdjustableDayLengthMono
